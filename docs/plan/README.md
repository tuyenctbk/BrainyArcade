# BrainyArcade Implementation Plan

BrainyArcade is a logic-focused game hub designed for Android TV, Android Auto, Mobile, and Tablet. The goal is to provide high-quality, brain-teasing puzzles with a unified input system.

## 1. Project Overview
- **Name:** Brainy Arcade
- **Target Platforms:** Android TV, Android Auto (parked), Mobile, Tablet.
- **Input Methods:** D-Pad (TV/Auto), Rotary (Auto), Touch (Mobile/Tablet).
- **Primary Tech:** Kotlin, View-based Canvas (for performance), leanback (for TV features).

## 2. Platform Strategy
| Platform | UI Priority | Input Focus |
|---|---|---|
| **Android TV** | 1080p/4K, Overscan safe, Immersive | D-Pad navigation |
| **Android Auto** | High contrast, large hit areas | Rotary selection, D-pad events |
| **Mobile/Tablet** | Adaptive layouts, Portrait/Landscape | Precise touch, gestures |

## 3. Core Architecture
- **`BaseGameActivity`**: Abstract class handling shared logic (analytics, help dialogs, sound management).
- **`GameView`**: Interface for game rendering logic.
- **`UnifiedInputManager`**: Map platform-specific events (Rotary, D-pad, Touch) to game commands (Up, Down, Select).
