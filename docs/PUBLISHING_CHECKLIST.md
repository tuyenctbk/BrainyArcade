# Publishing Checklist: BrainyArcade

## 1. Google Play Console Requirements
- [ ] **Store Listing**: Draft description with 15-language translation.
- [ ] **Icons**: 512x512 PNG, Adaptive Icon (Background/Foreground).
- [ ] **Feature Graphic**: 1024x500 PNG.
- [ ] **Screenshots**: TV (1080p), Mobile (7", 10"), and Android Auto.
- [ ] **Privacy Policy**: Link to public GitHub hosting or dedicated site.
- [ ] **App Bundle**: Release-signed .aab file.

## 2. GitHub Public Repository Setup
- [ ] **LICENSE**: Choose MIT or Apache 2.0.
- [ ] **README.md**: Comprehensive guide, screenshots, and architecture overview.
- [ ] **.gitignore**: Ensure no `local.properties` or keys are included.
- [ ] **Code Samples**: Document how to add a new game using the Base infrastructure.

## 3. Compliance & Legal
- [ ] **GDPR/CCPA**: Privacy policy explicitly mentioning Firebase/AdMob.
- [ ] **Age Rating**: Content rating survey for Google Play.
- [ ] **TV Compliance**: Verified `LEANBACK_LAUNCHER` and Banner assets.
